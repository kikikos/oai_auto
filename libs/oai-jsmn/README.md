JSMN
====

Commands
----------
```
$make 
$sudo make install
$sudo make uninstall
```
Usage: 
----------
After compile, copy libjsmn.a under /usr/lib and jsmn.h under /usr/local/include
```
$sudo cp libjsmn.a /usr/lib 
$sudo cp  jsmn.h  /usr/local/include
```

Example:
----------
#include <jsmn.h>
